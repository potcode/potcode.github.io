常用的css的一些属性总结

发表于 2017-02-06 |

总结了一些常用的css

1. user-select

user-select 是一个不规范的属性(unsupported Webkit property)，它没有出现在css的规范草案中。

作用：用来控制内容的可选择性

常见的值：

  • auto – 默认值，用户可以选中元素的内容
  • none – 用户不能选择元素中的任何内容
  • text – 用户可以选择元素中的文本
  • element – 文本可选，但是仅限于元素的边界内(目前只有IE和FF支持)
  • all – 在编辑器内，如果双击/上下文点击发生在子元素上，改值的最高级祖先元素将被选中
  • -moz-none– firefox 浏览器专有，元素和子元素的文本将不能被选。但是，子元素可以通过text重设回可以选择。

IE10也将支持该属性，Gecko和webkit也支持此属性，但是需要添加各自的私有前缀。Opera浏览器不支持。

2. -webkit-touch-callout

-webkit-touch-callout 是一个不规范的属性(unsupported Webkit property)，它没有出现在css的规范草案中。

作用：触摸并按住触摸目标时，禁止或者显示系统默认菜单

常见的值

  • none 系统默认菜单被禁用
  • inherit 系统默认菜单不被禁用

测试结果： -webkit-touch0-callout:none

  1. iOS 长按时不触发系统菜单
  2. iOS 和 android 长按时不触发下载图片菜单

3. 神奇的 :nth-child 属性

下面是我经常用到的一些实例（n都代表整数）

  1. :nth-child(n) 选取第n个标签
  2. :nth-child(n+m) 选取大于等于m的标签
  3. :nth-child(-n+m) 选取小于等于m的标签
  4. :nth-child(2n) 选取偶数标签
  5. :nth-child(2n+1) 先去奇数标签
  6. :nth-child(3n+1) “隔二选一”
  7. :first-child 选取第一个标签
  8. :last-child 选取第二个标签
  9. :nth-last-child(m) 选取倒数第m个标签
